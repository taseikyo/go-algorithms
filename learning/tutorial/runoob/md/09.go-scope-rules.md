# Go 语言变量作用域

[README](../README.md) | [Go 语言函数](08.go-functions.md) | [Go 语言数组](10.go-arrays.md)

Go 语言中变量可以在三个地方声明：

- 函数内定义的变量称为局部变量
- 函数外定义的变量称为全局变量
- 函数定义中的变量称为形式参数
